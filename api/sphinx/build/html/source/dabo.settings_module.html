Dabo Global Settings

Do not modify this file directly. Instead, create a file called settings_override.py, and copy/paste the settings section below into the settings_override.py file. This way, when you update Dabo, you won’t blow away your custom tweaks.

Note that creating a settings_override.py isn’t the only way to tweak the settings - your custom code can also just make the settings in the dabo namespace at runtime, eg:

import dabo
dabo.eventLogging = True
<do stuff>
dabo.eventLogging = False

Note

settings_override.py is not the appropriate place to put application-specific settings, although it may seem at first like an easy place to do so.
