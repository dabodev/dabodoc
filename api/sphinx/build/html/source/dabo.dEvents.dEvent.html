dEvents.dEvent - class¶

Base class for Dabo events.

Event objects are instantiated in self.raiseEvent(), and passed to all callbacks registered with self.bindEvent().

User code can define custom events by simply subclassing Event and then using self.bindEvent() and self.raiseEvent() in your objects.
