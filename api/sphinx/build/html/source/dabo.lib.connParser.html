connParser functions¶

This is the description of standalone Python functions in the dabo.lib.connParser module.

Functions Summary¶

Functions API¶

dabo.lib.connParser.createXML(cxns, encoding=None)¶

Returns the XML for the passed connection info. The info can either be a single dict of connection info, or a list/tuple of such dicts.

dabo.lib.connParser.fileRef(ref='')¶

Handles the passing of file names, file objects, or raw XML to the parser. Returns a file-like object, or None.

dabo.lib.connParser.genConnXML(d)¶

Receive a dict containing connection info, and return a ‘connection’ XML element.

dabo.lib.connParser.getConnTemplate()¶
dabo.lib.connParser.getXMLWrapper(encoding=None)¶
dabo.lib.connParser.importConnections(pth=None, useHomeDir=False)¶

Read the connection info in the file passed as ‘pth’, and return a dict containing connection names as keys and connection info dicts as the values.

If ‘useHomeDir’ is True, any file-based database connections will have their pathing resolved based upon the app’s current HomeDirectory. Otherwise, the path will be resolved relative to the connection file itself.
