dabo.dBug.loggit(fnc)¶

Decorator function to create a log of all methods as they are called. To use it, modify all your methods from:

def someMethod(...):

to:

@loggit def someMethod(...):

Be sure to add:

from dabo.dBug import loggit

to the import statements for every file that uses loggit. You can set the name and location of the log file by overriding the setting for dabo.loggitFile. By default, this value will be ‘functionCall.log’.
