dLocalize functions¶

This is the description of standalone Python functions in the dabo.dLocalize module.

|method_summary| Functions Summary¶

|API| Functions API¶

dabo.dLocalize.getDaboLocaleDir()¶
dabo.dLocalize.install(domain='dabo', localedir=None, unicode_mo=True)¶

Install the gettext translation service for the passed domain.

Either Dabo will be the only domain, or Dabo will be the fallback for a different domain that the user’s application set up.

dabo.dLocalize.isValidDomain(domain, localedir)¶

Return True if the localedir appears to contain translations for the domain.

dabo.dLocalize.n_(s)¶
dabo.dLocalize.setLanguage(lang=None, charset=None)¶

Change the language that strings get translated to, for all installed domains. NOTE: rather than call the install() method of the gettext.translation objects, which would globally bind the ‘_’ name, we’ll just set the ‘_currentTrans’ name to the translation object.
