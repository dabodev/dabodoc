DesignerUtils functions¶

This is the description of standalone Python functions in the dabo.lib.DesignerUtils module.

|method_summary| Functions Summary¶

|API| Functions API¶

dabo.lib.DesignerUtils.addCodeToClassDict(clsd, cd)¶

Takes the code that was stored in a separate file and re-integrates it into the class dict. Since this is a recursive structure, with children nested inside other children, it will be called recursively. No return value, as it modifies the class dict directly.

dabo.lib.DesignerUtils.addSizerDefaults(defaults)¶

Takes a dict of defaults, with the class as the key and the various defaults as the values. Used by external apps to customize behaviors for their own classes.

dabo.lib.DesignerUtils.getCodeObjectSeperator()¶
dabo.lib.DesignerUtils.getDefaultSizerProps(cls, szType)¶
dabo.lib.DesignerUtils.getSizerDefaults()¶

Return a dict that contains the defaults for the various controls based upon what sort of sizer they are contained within.

dabo.lib.DesignerUtils.parseCodeFile(txt)¶

This method takes the content of a code file saved by the class designer and parses it into a dict containing the code-ID values as keys, and a dict containing the methods as the values.
