dabo.dColors.colorNameFromTuple(colorTuple, firstOnly=False)¶

Returns a list of color names, if any, whose RGB tuple matches the specified tuple. If ‘firstOnly’ is True, then a single color name will be returned as a string, not a list; the string will be empty if there is no match.

dabo.dColors.colorTupleFromHex(hx)¶
dabo.dColors.colorTupleFromName(color)¶

Given a color name, such as “Blue” or “Aquamarine”, return a color tuple.

This is used internally in the ForeColor and BackColor property setters. The color name is not case-sensitive. If the color name doesn’t exist, an exception is raised.
